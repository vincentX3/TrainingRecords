# TrainingRecords

## What's this?
出于个人健身的需要，开发一个轻量的app记录每日训练内容。
目标：记录个人每日运动状况，兼具数据统计分析功能
功能点：
- 每日记录，动作，动作等级（重量，细分动作等），数量
- 统计分析，按照时间／动作，统计数量
